# AdventureWorksDW Market Basket Analysis

This project is a beginner-friendly yet professional data analysis workflow built on the **AdventureWorksDW** dataset, a popular OLAP-style dataset commonly used for BI and analytics practice.  
The goal is to simulate a real-world analytics scenario in **E-commerce / Retail** contexts by performing exploratory data analysis (EDA) and producing actionable business insights.

##  Business Problem

The Sales & Product Analytics team wants to understand **overall sales performance**, **customer behavior**, and **product-level trends** based on historical transaction data.  
The insights from this analysis will support:

- Forecasting demand  
- Optimizing inventory  
- Improving customer segmentation  
- Identifying best-selling and underperforming products  
- Understanding seasonality in sales  

This type of analysis is central to both **E-commerce/Retail** and **SaaS/Product Analytics**, making it a strong foundation for portfolio projects.
